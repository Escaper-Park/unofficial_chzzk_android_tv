# 채널 기능
(언)팔로우, 라이브 방송 보기, 동영상 보기 기능을 사용할 수 있습니다.

상단의 팔로우 버튼을 누르면 팔로우 할 수 있습니다. 이미 팔로우 되어있는 채널이라면 언팔로우 알림 메시지가 뜨게 됩니다.

## 라이브 방송
해당 채널이 라이브 방송을 하고 있다면 라이브 목록에 나타납니다.
