# UnOfficial Chzzk for Android TV

**비공식** [치지직](https://chzzk.naver.com/) 안드로이드 스마트 TV 애플리케이션입니다. 이 애플리케이션은 NAVER, 치지직과 관련이 없습니다.

> [!NOTE]
>
> 문의 및 제보: escaper.bgp@gmail.com

> [!CAUTION]
>
> 보안을 위해 이 [GitHub 레포지토리](https://github.com/Escaper-Park/unofficial_chzzk_android_tv) 이외의 웹사이트에서 APK 파일을 다운로드 받지 마세요.
>
> 이 Repository의 코드는 개인적인 학습 및 사용 목적으로만 수정하여 사용할 수 있습니다. 수정된 버전으로 프로그램을 배포하는 행위는 금지됩니다.

### [설치 방법](documents/INSTALL.md)
> [!CAUTION]
>
> 기존에 설치된 비공식 치지직 안드로이드 TV앱이 있다면 삭제한 뒤 설치해주세요.

> [!NOTE]
>
> 설치에 문제가 있을 경우 Android TV의 개발자 모드를 켠 뒤 설치를 진행합니다.

APK 파일은 USB 저장소, 네트워크 공유, [`파일 관리자+`](https://play.google.com/store/apps/details?id=com.alphainventor.filemanager&hl=ko), [`Send Files To TV`](https://play.google.com/store/apps/details?id=com.yablio.sendfilestotv&hl=en_US)와 같은 파일 전송 앱을 사용해 TV로 옮길 수 있습니다.

## 앱 사용 방법

리모컨 6개 ( :arrow_up: 위, :arrow_down: 아래, :arrow_left: 좌, :arrow_right: 우, :ok: 확인, :leftwards_arrow_with_hook: 뒤로가기 ) 버튼을 사용하여 조작합니다.

화면별 사용 방법을 확인해주세요.
